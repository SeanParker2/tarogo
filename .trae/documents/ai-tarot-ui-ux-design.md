# AI塔罗牌占卜小程序 - UI/UX设计文档

## 1. 设计原则

### 1.1 视觉设计原则
- **神秘感**：营造塔罗牌特有的神秘氛围
- **现代感**：结合AI技术的现代科技感
- **亲和力**：界面友好，降低用户心理门槛
- **一致性**：保持整体视觉风格统一

### 1.2 交互设计原则
- **简洁性**：操作流程简单明了
- **反馈性**：每个操作都有明确反馈
- **容错性**：允许用户撤销或重新选择
- **引导性**：清晰的步骤引导

## 2. 色彩系统

### 2.1 主色调
- **主色**：神秘紫 (#6B46C1)
- **辅助色**：金色 (#FFD700)
- **背景色**：深紫 (#4C1D95)
- **文字色**：浅灰 (#F3F4F6)

### 2.2 状态色
- **成功**：翡翠绿 (#10B981)
- **警告**：琥珀色 (#F59E0B)
- **错误**：玫瑰色 (#EF4444)
- **信息**：天蓝色 (#3B82F6)

### 2.3 渐变效果
```css

.starfield {

}
```

## 6. 交互设计

### 6.1 抽牌交互流程
1. **洗牌阶段**
   - 点击开始洗牌
   - 卡牌背面旋转动画
   - 随机打乱效果
   - 完成提示

2. **切牌阶段**
   - 滑动切牌手势
   - 卡牌分组动画
   - 选择牌堆

3. **抽牌阶段**
   - 点击抽取每张牌
   - 3D翻牌动画
   - 牌面展示效果

### 6.2 动画效果设计

#### 加载动画
```css
@keyframes shuffle {
    0% { transform: rotate(0deg) translateX(0); }
    25% { transform: rotate(-5deg) translateX(-10px); }
    50% { transform: rotate(5deg) translateX(10px); }
    75% { transform: rotate(-3deg) translateX(-5px); }
    100% { transform: rotate(0deg) translateX(0); }
}

@keyframes glow {
    0% { box-shadow: 0 0 5px rgba(255, 215, 0, 0.5); }
    50% { box-shadow: 0 0 20px rgba(255, 215, 0, 0.8); }
    100% { box-shadow: 0 0 5px rgba(255, 215, 0, 0.5); }
}
```

#### 翻牌动画
```css
@keyframes flipCard {
    0% { transform: rotateY(0deg); }
    50% { transform: rotateY(90deg); }
    100% { transform: rotateY(0deg); }
}
```

## 7. 响应式设计

### 7.1 适配方案
- **基础尺寸**：750rpx (iPhone 6)
- **适配策略**：rpx单位 + 百分比布局
- **断点设置**：
  - 小屏设备 (< 375px)
  - 标准设备 (375px - 414px)
  - 大屏设备 (> 414px)

### 7.2 组件适配
```css
/* 卡片容器 */
.card-container {
    width: 90%;
    max-width: 700rpx;
    margin: 0 auto;
}

/* 网格布局 */
.grid-layout {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200rpx, 1fr));
    gap: 20rpx;
```

## 11. 设计规范文档

### 11.1 组件库规范
- **按钮组件**：主按钮、次按钮、文字按钮
- **卡片组件**：信息卡片、结果卡片
- **输入组件**：文本输入、选择器
- **反馈组件**：提示框、加载框

### 11.2 间距规范
- **基础间距**：8px、16px、24px、32px
- **组件间距**：垂直24px，水平16px
- **页面边距**：左右32px，上下24px

### 11.3 圆角规范
- **小圆角**：4px (按钮、标签)
- **中圆角**：8px (卡片、输入框)
- **大圆角**：16px (大卡片、弹窗)
